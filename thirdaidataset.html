class thirdai.dataset.CSVDataSource¶

Bases: DataSource

CSV data source that can be used to load from a cloud storage instance such as s3 and GCS.

Parameters:

  • storage_path – Path to the CSV file.

  • batch_size – Batch size

  • gcs_credentials_path – Path to a file containing GCS credentials. This is typically a credentials.json file. For the authorization protocol to work, the credentials file must contain a project ID, client E-mail, a token URI and a private key.

Note: To read a file from s3, Pandas will expect a credentials file

containing an AWS access key id and an AWS secret key.
