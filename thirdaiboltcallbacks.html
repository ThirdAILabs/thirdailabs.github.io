class thirdai.bolt.callbacks.EarlyStopCheckpoint

Bases: Callback

__init__(monitored_metric: str, model_save_path: str, patience: int, min_delta: float) → None

This callback is intended to stop training early based on prediction results from a given validation set. Saves the best model to model_save_path. :param * monitored_metric: The metric to monitor for early stopping. The metric is assumed to be associated with validation data. :param * model_save_path: string. The file path to save the model that scored the best on the validation set :param * patience: int. The nuber of epochs with no improvement in validation score after which training will be stopped. :param * min_delta: float. The minimum change in the monitored quantity to qualify as an improvement, i.e. an absolute change of less than min_delta will count as no improvement.
